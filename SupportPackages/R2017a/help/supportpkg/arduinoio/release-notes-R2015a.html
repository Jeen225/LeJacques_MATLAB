Support for Arduino Leonardo and other boards with MATLAB Support Package for Arduino Hardware

You can now use these boards with MATLAB® Support Package for Arduino® Hardware:

  • Arduino Leonardo

  • Arduino Micro

  • Arduino Mega ADK

  • Arduino Nano 3.1

  • Sparkfun Redboard

  • Sparkfun Digital Sandbox

  • Sainsmart Uno

  • Sainsmart Mega 2560

New configurePin function

In release R2015a, use configurePin to set your pin mode. You can use string values, such as ‘D2' to specify digital pins.
