Install Hardware Driver

Install Arduino Driver on a Windows System

  1. Before installing device drivers, make sure that you have administrative privileges.

  2. Open Device Manager, and find your device mentioned as USB Serial Device or Unknown Device.

  3. Right-click on the device, and select Update Driver Software.

  4. Select Browse my computer for driver software.

  5. Enter the path of the Arduino® IDE in your computer. To obtain the path, type arduinoio.IDEroot in the MATLAB® Command Window.

  6. Click Install to install the driver software for Arduino board.

  7. You receive confirmation of a successful installation.

Install FTDI Driver

If you do not have a FTDI driver installed on your system, follow one of these instructions:

  • Connect your system to the Internet, and then connect an FTDI device. If your system is connected to the Internet, the FTDI driver installs automatically once the FTDI device is plugged-in.

  • Follow the installation guides to manually install the FTDI drivers.
