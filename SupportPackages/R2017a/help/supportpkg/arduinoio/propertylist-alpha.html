MATLAB Support Package for Arduino Hardware Properties

Alphabetical List By Category

A

Address Display I2C device bus address
AvailableAnalogPins Display analog pins available on Arduino device
AvailableDigitalPins Display digital pins available on Arduino device
AvailablePins Display pins available on Arduino device

B

BitOrder Specify SPI communication bit order
Board Specify Arduino hardware type
Bus Specify I2C bus number

C

ChipSelectPin Pin connected to chip select line on an SPI device

L

Libraries Display installed Arduino add-on libraries

M

MaxPulseDuration Specify maximum pulse duration
MinPulseDuration Specify minimum pulse duration
Mode Display SPI communication mode

P

Pins Display Arduino device pins
Port Display Arduino communication port to host computer
