dcmotor

Create DC motor connection

collapse all in page

Syntax

dcm = dcmotor(dev,motornum)
dcm = dcmotor(dev,motornum,Name,Value)

Description

example

dcm = dcmotor(dev,motornum) creates a DC motor connection with the specified motor number on an Adafruit® Motor Shield connected to an Arduino® board.

example

dcm = dcmotor(dev,motornum,Name,Value) also creates a DC motor connection with additional options specified by one or more Name,Value pair arguments.

Input Arguments

collapse all

dev — Adafruit Motor Shield connection
adafruit motor shield object

Adafruit Motor Shield connection, created using addon function, specified as an adafruit motor shield object.

motornum — Port number on shield
numeric

Port number on the shield that the motor is connected to, specified as a number. Valid port values are 1, 2, 3, and 4.

Name-Value Pair Arguments

Specify optional comma-separated pairs of Name,Value arguments. Name is the argument name and Value is the corresponding value. Name must appear inside single quotes (' '). You can specify several name and value pair arguments in any order as Name1,Value1,...,NameN,ValueN.

Use Name,Value arguments to specify parameter initial values and additional Properties of dcmotor function during model creation. For example, dcm = dcmotor(dev,1,'Speed',0.2); creates a dc motor connection with the Speed property set to 0.2.

Properties

collapse all

DC motor object properties include:

Speed — DC motor speed
0 (default) | numeric

DC motor speed specified as the comma-separated pair consisting of 'Speed' and a number between —1 and 1. You can specify motor speed when you create the dcmotor object. Also, you can change the motor speed after creating the dcmotor object. For example, 

a = arduino('COM4', 'Uno','Libraries', 'Adafruit/MotorShieldV2');
dev = addon(a,'Adafruit/MotorshieldV2');
dcm = dcmotor(dev, 1);
% Change the speed to 0.2
dcm.Speed = 0.2

Pins — Pins used by dcmotor object
character vector

Pins used by dcmotor object, represented in alpha-numeric values. Pins is a read-only property.

MotorNumber — DC motor port number
numeric

DC motor port number on the shield where the DC motor is connected to, specified during the dcmotor object creation. MotorNumber is a read-only property.

IsRunning — DC motor state
0 (default) | 1

DC motor state, indicated as a boolean value. IsRunning is 0 if the motor has not started and 1 otherwise. IsRunning is a read-only property.

Output Arguments

collapse all

dcm — DC motor object
dcmotor object

DC motor connection, returned as a dcmotor object.

Examples

collapse all

Create a DC Motor Connection

Create a DC motor connection to an Adafruit Motor Shield attached to an Arduino hardware.

Connect to Arduino hardware, and create an add-on connection to an Adafruit Motor Shield.

a = arduino('COM7','Uno','Libraries','Adafruit/MotorShieldV2');
dev = addon(a,'Adafruit/MotorShieldV2');

Create a DC motor connection at port 1.

dcm = dcmotor(dev,1)
dcm = 

  dcmotorv2 with properties:

    MotorNumber: 1 (M1)
          Speed: 0
      IsRunning: 0

Specify Speed for a DC Motor Connection

Create a DC motor connection at port 1 on the shield, and set the speed to 0.2.

dcm = dcmotor(dev,1,'Speed',0.2)
dcm = 

  dcmotorv2 with properties:

    MotorNumber: 1 (M1)
          Speed: 0.20
      IsRunning: 0
