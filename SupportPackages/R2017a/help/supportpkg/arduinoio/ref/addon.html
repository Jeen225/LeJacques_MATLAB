addon

Create add-on device connection to Adafruit Motor Shield V2

collapse all in page

Syntax

motor = addon(a,'Adafruit\MotorshieldV2')
motor = addon(a,'Adafruit\MotorshieldV2',Name,Value)

Description

example

motor = addon(a,'Adafruit\MotorshieldV2') creates an add-on connection to Adafruit® Motor Shield V2 connected to the Arduino® hardware a. Also, you can create additional add-on libraries such as L3G, LSM303, 1–Wire, HC-SR04, and Capacitive sensor. Furthermore, you can also create your own custom Arduino add-on libraries as mentioned in Create Custom Arduino Add-On Device, Library, or Code.

example

motor = addon(a,'Adafruit\MotorshieldV2',Name,Value) creates an add-on connection to Adafruit Motor Shield V2 with additional options specified by one or more Name,Value pair arguments.

Input Arguments

collapse all

a — Arduino hardware connection
arduino object

Arduino hardware connection created using arduino, specified as an arduino object.

Name-Value Pair Arguments

Specify optional comma-separated pairs of Name,Value arguments. Name is the argument name and Value is the corresponding value. Name must appear inside single quotes (' '). You can specify several name and value pair arguments in any order as Name1,Value1,...,NameN,ValueN.

Use Name,Value arguments to specify parameter initial values and additional Properties of addon function during model creation. For example, motor = addon(a, ‘Adafruit\MotorshieldV2',‘I2CAddress', ‘0x07' ‘pwmfrequency', 3200); creates an adafruit motor shield connection with the I2CAddress property set to 0x07 and the PWMFrequency property set to a value of 3200.

Properties

collapse all

Adafruit motor shield object properties include:

I2CAddress — I2C device address
90 (0x60) (default) | numeric | character vector

I2C address, specified as the comma-separated pair consisting of 'I2CAddress' and its value. The value can either a hexadecimal value or a scalar integer. I2CAddress is a read-only property whose value is specified only during adafruit motor shield object creation.

Example: shield = addon(a,'Adafruit\MotorshieldV2','I2CAddress','0x61'); sets the I2C address to 97 or 0x61.

PWMFrequency — PWM frequency of Adafruit Motor Shield V2
1600 (default) | numeric

PWM frequency of Adafruit Motor Shield V2, specified as the comma-separated pair consisting of 'PWMFrequency' and a number. PWMFrequency is a read-only property whose value is specified only during adafruit motor shield object creation.

Example: shield = addon(a,'Adafruit\MotorshieldV2','PWMFrequency',3200); sets the PWM frequency to 3200.

Pins — Arduino pins used by Adafruit Motor Shield V2
96 (default) | numeric | character vector

Arduino pins used by Adafruit Motor Shield V2, represented in alpha-numeric values. Pins is a read-only property.

You can set the properties of the adafruit motor shield object only during its object creation. After creating the adafruit motor shield object, you can use dot notation to access its property values. For example, 

motor = addon(a,'Adafruit\MotorshieldV2');
motor.I2CAddress
ans = 
96

Output Arguments

collapse all

motor — Adafruit Motor Shield V2 connection
adafruit motor shield object

Adafruit Motor Shield V2 connection, returned as an adafruit motor shield object.

Examples

collapse all

Create a Motorshieldv2 Object

Connect to Arduino hardware, and create an add-on connection to an Adafruit Motor Shield.

a = arduino('COM7','Uno','Libraries','Adafruit/MotorShieldV2');
motor = addon(a,'Adafruit\MotorshieldV2')
motor = 

motorshieldv2 with properties:

            Pins: A4(SDA), A5(SCL)
      I2CAddress: 96 (0x60)
    PWMFrequency: 1600 (Hz)

Specify I2C Address and PWM Frequency

Connect to Arduino hardware, and create an add-on connection to an Adafruit Motor Shield with an I2C address of 0x61 and PWM frequency of 1200.

a = arduino('COM7','Uno','Libraries','Adafruit/MotorShieldV2');
motor = addon(a,'Adafruit\MotorshieldV2','I2CAddress','0x61', 'PWMFrequency',1200)
motor = 

motorshieldv2 with properties:

            Pins: A4(SDA), A5(SCL)
      I2CAddress: 97 (0x61)
    PWMFrequency: 1200 (Hz)
